# 🚇 서울교통공사 지하철 혼잡도 대시보드

서울 지하철 혼잡도 데이터를 시각화하고 분석하는 Streamlit 대시보드입니다.

## 📊 프로젝트 소개

서울교통공사의 지하철 혼잡도 데이터(2025년 9월 30일 기준)를 활용하여 노선별, 역별, 시간대별 혼잡도 패턴을 분석하고 시각화합니다.

## 🚀 시작하기

### 1. 의존성 설치

```bash
pip install -r requirements.txt
```

### 2. 애플리케이션 실행

```bash
streamlit run app.py
```

브라우저에서 자동으로 `http://localhost:8501`이 열립니다.

## 📁 프로젝트 구조

```
c:\bread\

│   ├── data.py                               # 데이터 로딩/전처리

    ├── 02_노선별.py                           # 노선 단위 분석
    └── 03_역상세.py                           # 역별 상세 분석
```

## ✨ 주요 기능

### 📊 개요 페이지
- 전체 혼잡도 KPI (평균, 최대, 피크 시간대)
- 시간대별 평균 혼잡도 추이 차트
- 혼잡 TOP 10 역 순위
- 호선별/방향별 통계

### 🚉 노선별 페이지
- 노선 선택 및 요약 통계
- 시간대별 평균 혼잡도 (방향별 비교)
- 역별 피크 혼잡도 순위 테이블
- 특정 시간대 역별 비교 바 차트
- 역별 시간대 히트맵

### 🔍 역상세 페이지
- 역 검색 및 선택
- 방향별 요약 카드
- 시간대별 혼잡도 라인 차트 (피크 구간 강조)
- 가장 혼잡한 시간대 TOP 3
- 상세 데이터 테이블 및 CSV 다운로드
- 시간대별 방향 비교 히트맵

## 🎨 필터 기능

모든 페이지에서 다음 필터를 사용할 수 있습니다:
- 요일 구분 (평일/토요일/일요일)
- 호선 선택 (다중 선택 가능)
- 역 선택 (호선에 연동)
- 방향 선택 (상선/하선, 내선/외선)
- 시간대 범위 슬라이더
- 혼잡도 범위 필터 (고급)

## 📈 데이터 정보

- **출처**: 서울교통공사
- **기준일**: 2025년 9월 30일
- **포함 내용**: 1~8호선 지하철 혼잡도
- **시간대**: 05:30 ~ 00:30 (30분 간격)
- **데이터 건수**: 1,673개 (요일 x 호선 x 역 x 방향)

## 💡 혼잡도 해석

- 🟢 **여유** (0-30%): 좌석이 여유롭습니다
- 🟡 **보통** (30-70%): 일부 좌석이 찼습니다
- 🟠 **혼잡** (70-130%): 대부분 좌석이 차고 서 있는 승객이 있습니다
- 🔴 **매우혼잡** (130%+): 승하차가 어려울 정도로 혼잡합니다

## 🛠️ 기술 스택

- **Streamlit**: 대시보드 프레임워크
- **Pandas**: 데이터 처리 및 분석
- **Plotly**: 인터랙티브 차트 시각화
- **Python 3.8+**: 프로그래밍 언어
